## 接口文档



### 1. 获取首页轮播图照片

#### 方法类型&路径

>GET		/api/rotationchart

#### 传入参数

>无

#### 返回值

```js
{
  imgId:0,
  imgURL:""
}
```



### 2. 获取首页最新资讯信息

#### 方法类型&路径

>GET		/api/latestinfo

#### 传入参数

> 无

#### 返回值

```js
{
  latestInfoList:
    [
      {
        itemID:0,
        itemHeader:"标题1",//最新资讯的标题
        itemTime:"2021年7月12日",
        itemImg:"/a"//图片URL,

      },
      {
        itemID:1,
        itemHeader:"标题2",//最新资讯的标题
        itemTime:"2021年7月13日",
        itemImg:"/b"//图片URL
      }，
     
    ] 
}
```



### 3. 获取每一条最新资讯的详细信息

#### 方法类型&路径

>GET		/api/latestinfo/result?itemid=123456



#### 参入参数

>无

#### 返回值

```js
{
  itemInfo:{
    header:"调研标题",
    time:"2020年7月23日",
    content:"文章内容",
    imgList:[  //文章图片URL
      "/a",
      "/b",
      "/c"
    ]，
    video:"/d"
  }
}
```
